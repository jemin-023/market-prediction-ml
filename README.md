# 📈 Market Prediction & Portfolio Optimization Sandbox

> **Project Status:** 🚧 Active Experimentation & Development 🚧

## 🎯 Overview
This repository serves as an experimental sandbox for algorithmic trading, machine learning, and quantitative finance. Instead of relying on a single monolithic model, the goal of this project is to build, test, and evaluate **multiple trading strategies** and use quantitative methods to **optimize a unified portfolio**.

Currently, this project is in its foundational stages. The focus is on building robust backtesting infrastructure, gathering clean data, and exploring various Alpha-generating ideas.

## 🔬 Planned Experiments & Strategies
As the project evolves, I plan to experiment with the following concepts:

* **Statistical Arbitrage & Mean Reversion:** Testing classic quantitative strategies.
* **Trend Following & Momentum:** Capturing macroeconomic and sector-wide trends.
* **Machine Learning Signals:** Using tree-based models (LightGBM/XGBoost) and neural networks for price and volatility forecasting.
* **Alternative Data / NLP:** Extracting sentiment from financial news and earnings call transcripts.
* **Portfolio Optimization:** Utilizing Modern Portfolio Theory (MPT), Hierarchical Risk Parity (HRP), and Kelly Criterion to dynamically allocate capital across different active strategies.
